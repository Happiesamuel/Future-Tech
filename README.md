## 🧠 About

**Future Technologies** is a modern, high-performance web platform built to deliver stunning digital experiences using cutting-edge tools and clean UI/UX principles.

---

## ✨ Features

* 🎯 Modern UI/UX design
* 🎞️ Smooth animations with GSAP
* ⚡ Optimized performance (Next.js)
* 📱 Fully responsive design
* 🧩 Scalable architecture
* 🎨 Tailwind + shadcn/ui styling

---

## 🛠️ Tech Stack

| Technology   | Description    |

| GSAP         | Animations     |
